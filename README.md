# 🌟 SkinDoc-AI

## AI Skin Detection & Personalized Skincare Recommender

SkinDoc-AI is a deep learning-powered web application that analyzes facial skin conditions and provides personalized skincare recommendations. The system uses computer vision and image classification techniques to identify common skin conditions and suggest suitable skincare guidance.

The application is deployed on Hugging Face Spaces, allowing users to perform skin analysis directly through a web browser.

---

## 🚀 Features

- AI-powered skin condition detection
- Real-time facial image capture
- Personalized skincare recommendations
- User-friendly web interface
- Instant prediction results
- Browser-based access through Hugging Face Spaces

---

## 🔍 Detectable Skin Conditions

The model can identify the following skin categories:

- Acne
- Dry Skin
- Oily Skin
- Hyperpigmentation
- Normal Skin

---

## 📋 How to Use

1. Open the application.
2. Click **Capture Your Face**.
3. Ensure your face is clearly visible and centered.
4. Use good lighting for better prediction accuracy.
5. Capture the image.
6. Wait for the AI model to analyze your skin.
7. View the detected skin condition and personalized skincare recommendations.
